# Контент-план канала

## Целевые аудитории и БМПХС

### Сегмент А — Предприниматели малого бизнеса (ИП, микробизнес)
_35-50 лет, уже работают, но хотят расти без раздутого штата_

| | |
|---|---|
| **Боли** | Нет времени на маркетинг. Всё сам. Нанять дорого, учиться некогда. |
| **Мечты** | Бизнес работает, а я занимаюсь тем, что люблю. Клиенты приходят сами. |
| **Проблемы** | Telegram-канал ведётся хаотично или не ведётся совсем. Не знают что писать. |
| **Хотелки** | Готовые инструменты, шаблоны, "сделай за меня но в моём стиле". |
| **Страхи** | AI напишет роботизированно. Клиенты заметят. Потеряю голос. |

---

### Сегмент Б — Эксперты и консультанты (коучи, психологи, юристы)
_28-45 лет, сильная экспертиза, слабый маркетинг_

| | |
|---|---|
| **Боли** | Много знаю, но не умею упаковать. Пишу пост 3 часа. |
| **Мечты** | Клиенты приходят по рекомендации + из канала. Меня знают как эксперта. |
| **Проблемы** | ChatGPT пишет не моим языком. Стыдно публиковать. |
| **Хотелки** | Инструмент который учится у меня, а не у интернета. |
| **Страхи** | Выгляжу ненастоящей. Аудитория уйдёт если узнает что AI. |

---

### Сегмент В — Технари-одиночки (разработчики, фрилансеры)
_25-40 лет, умеют делать, не умеют продавать себя_

| | |
|---|---|
| **Боли** | Делаю крутые вещи, но никто не знает. Продажи через "сарафан" не масштабируются. |
| **Мечты** | Канал работает как воронка. Заказы без холодных переговоров. |
| **Проблемы** | Не знают о чём писать. Боятся выглядеть самозванцем. |
| **Хотелки** | Писать быстро, не выгорать, видеть результат. |
| **Страхи** | Напишу что-то — осудят. Лучше молчать. |

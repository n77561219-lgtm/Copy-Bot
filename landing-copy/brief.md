## 1. ЦЕЛЬ И АУДИТОРИЯ

### Цель сайта

Конвертировать посетителя в пользователя Telegram-бота @CopyBot.
**Целевое действие:** нажатие на кнопку "Попробовать бесплатно", ведущую в Telegram.
Промежуточная цель — провести посетителя через полный путь: осознание боли → понимание решения → доверие → решение.

Никаких форм, email-сборов, регистраций на сайте. Только одна кнопка → Telegram.

### Целевая аудитория

**Первичная:**
- Администраторы Telegram-каналов (1 000–100 000 подписчиков)
- Владельцы сообществ и групп ВКонтакте
- Авторы и создатели контента в MAX
- Ниши: финансы, лайфстайл, образование, бизнес, технологии, здоровье
- Боль: нет времени / сил на регулярный контент сразу на нескольких площадках
- Уже пробовали ChatGPT — результат не под формат платформы

**Вторичная:**
- SMM-менеджеры, ведущие 3–10 клиентских аккаунтов на разных платформах
- Инфобизнесмены, которым нужен контент для прогрева на всех площадках
- Блогеры, масштабирующие присутствие: Telegram + ВКонтакте + MAX одновременно

**Психографика:**
- Знают, о чём хотят писать — не знают как адаптировать под каждую площадку
- Ценят своё время выше денег
- Скептически настроены к AI ("это будет безликий текст не в том формате")
- Принимают решение быстро — если за 30 секунд непонятно, уходят
- Открывают лендинг на телефоне из Telegram-ссылки или рекламы ВКонтакте

## 2. ОФФЕР И ПОЗИЦИОНИРОВАНИЕ

### Главный оффер

> **"Посты для Telegram, ВКонтакте и MAX — за 3 минуты. Один бот. Все площадки."**

**Формула:**
`[Конкретный результат] + [Временной фрейм] + [Уникальный механизм] + [Устранение страха]`

Работаешь в одном месте — Telegram. Бот сам форматирует контент под правила каждой платформы.

### Главный дифференциатор

Два уровня уникальности:

1. **Мультиплатформенность:** Copy-Bot генерирует контент для **Telegram, ВКонтакте и MAX** — правильно отформатированный под каждую площадку. Один запрос → три адаптированных поста.

2. **Telegram-native interface:** Бот работает прямо внутри Telegram. Ни один конкурент (Jasper, Copy.ai, Writesonic) не даёт ни одного из этих двух преимуществ. Упоминать в каждой секции.

**Ключевое сообщение:** "Работаешь в Telegram — контент для всех площадок."

### Форматы по платформам

| Платформа | Особенности формата | Что учитывает бот |
|-----------|--------------------|--------------------|
| **Telegram** | Короткие абзацы, **bold**, эмодзи, без хэштегов | Длина, структура, channel voice |
| **ВКонтакте** | Длиннее, хэштеги, @упоминания, абзацы с отступами | SEO-хэштеги, ВК-форматирование |
| **MAX** | Короткий и цепляющий формат, визуальный акцент | Ритм, лаконичность, заголовок |

### Позиционирование против ChatGPT

| ChatGPT | Copy-Bot |
|---------|----------|
| Полный промпт каждый раз | Тема → пост в 2 клика |
| Не знает форматы платформ | Создан для TG, ВК и MAX |
| Generic, robotic output | Учитывает тон, нишу и площадку |
| Нет памяти между сессиями | Контекст канала сохраняется |
| Один пост — вручную адаптировать под каждую площадку | Один запрос → три варианта |
| $20/мес "для всего" | Специализированный инструмент |

**Ключевая фраза для FAQ и features:**
> "ChatGPT может написать пост. Но ты должен знать как попросить, отредактировать, отформатировать под Telegram, адаптировать для ВКонтакте, переписать для MAX — и повторить это завтра. Copy-Bot делает всё это за тебя."

## 3. СТИЛЬ ДИЗАЙНА

### Настроение и ощущение

**Dark premium SaaS** — профессиональный AI-инструмент, не игрушка.
Ощущение: "Это серьёзный инструмент для людей, которые серьёзно относятся к своему каналу."
Антипод: яркий, кричащий, "стартаперский" дизайн.

### Цветовая палитра

```
Фон (background):     #09090b  — near-black, не чёрный (глубина)
Поверхность (surface): #18181b  — карточки, секции
Акцент primary:        #7c3aed  — electric violet (AI, creativity)
Акцент glow:           rgba(124, 58, 237, 0.35)  — свечение за элементами
Текст primary:         #fafafa  — почти белый
Текст secondary:       #a1a1aa  — серый для подзаголовков и мелкого текста
Граница (border):      rgba(255, 255, 255, 0.08)  — еле заметная
Успех/акцент 2:        #10b981  — зелёный для checkmarks в pricing
```

**Mesh gradient для hero background:**
```css
background-image:
  radial-gradient(at 20% 30%, #4f46e5 0px, transparent 55%),
  radial-gradient(at 80% 10%, #7c3aed 0px, transparent 50%),
  radial-gradient(at 60% 80%, #0ea5e9 0px, transparent 45%);
```

### Типографика

| Элемент | Шрифт | Размер | Вес | Трекинг |
|---------|-------|--------|-----|---------|
| Hero headline | Satoshi или Cabinet Grotesk | `clamp(2.8rem, 7vw, 5.5rem)` | 800 | -0.04em |
| Секционные заголовки (H2) | Satoshi | `clamp(1.8rem, 4vw, 2.8rem)` | 700 | -0.02em |
| Подзаголовки (H3) | Satoshi | 1.1rem | 600 | 0 |
| Body text | Inter | 1rem | 400 | 0 |
| Labels, caps | Inter | 0.75rem | 600 | 0.08em, UPPERCASE |
| Pricing price | Satoshi | 2.5rem | 800 | -0.03em |
| Code / terminal | JetBrains Mono | 0.9rem | 400 | 0 |

**Источники шрифтов:**
- Satoshi, Cabinet Grotesk → fontshare.com (бесплатно, без ограничений)
- Inter → fonts.google.com
- JetBrains Mono → fonts.google.com

**Принцип:** Hero headline максимально большой и tight. Body text маленький и просторный. Контраст размеров создаёт иерархию.

### Иконки

Lucide Icons или Heroicons — outline style, 1.5px stroke, 24px base size.
Не использовать emoji как иконки в UI (только в контентных примерах постов).

### Borders и Shadows

```css
/* Карточки */
border: 1px solid rgba(255, 255, 255, 0.08);
border-radius: 16px;

/* Featured карточка (pricing) */
border: 2px solid #7c3aed;
border-radius: 20px;

/* Glow за product mockup */
box-shadow:
  0 0 0 1px rgba(255,255,255,0.08),
  0 32px 64px rgba(0,0,0,0.5),
  0 0 120px rgba(124,58,237,0.2);

/* CTA кнопка */
box-shadow: 0 0 24px rgba(124,58,237,0.4), 0 0 80px rgba(124,58,237,0.15);
```

## 6. ОБЯЗАТЕЛЬНЫЕ ЭЛЕМЕНТЫ И АНИМАЦИИ

### Кнопки

| Тип | Стиль | Текст | Действие |
|-----|-------|-------|----------|
| Primary CTA | `bg: #7c3aed`, glow, radius 10px, padding 14px 28px | "Попробовать бесплатно →" / "Открыть Copy-Bot в Telegram →" | `href="https://t.me/CopyBot"` |
| Secondary | Border `#7c3aed`, transparent bg, same radius | "Попробовать бесплатно →" | То же |
| Ghost | Border `rgba(255,255,255,0.15)`, transparent | "Выбрать план" | Scroll to pricing |
| Pricing Featured | Primary style + larger + glow | "Выбрать Create" | Открыть бота |

**Hover-состояние primary:** brightness 110%, box-shadow усиливается.
**Active-состояние:** scale(0.97), быстро (100ms).

### Анимации

| Элемент | Анимация | Техника |
|---------|----------|---------|
| Phone mockup в hero | Typewriter диалог, loop 5s | CSS keyframes + JS setInterval |
| Mesh gradient blobs | Медленный drift 8s loop | CSS @keyframes, transform |
| Секции при скролле | Fade + slide up на входе во viewport | CSS `animation-timeline: view()` |
| Feature cards hover | 3D tilt (max 8°) + surface shimmer | JS mousemove + CSS transform |
| Pricing featured card | Subtle pulse на border (2s loop) | CSS @keyframes opacity |
| Demo typewriter | Символы появляются 30ms/символ при входе в секцию | JS IntersectionObserver + setInterval |
| Marquee | Infinite horizontal scroll | CSS @keyframes, `width: max-content` |
| FAQ accordion | Smooth height expand | CSS max-height transition |
| Nav background | Glassmorphism при scroll > 80px | JS scroll event + CSS transition |
| Floating CTA | Появляется после scroll 30% | JS IntersectionObserver |

**Важно:** Все анимации должны уважать `prefers-reduced-motion: reduce` — при включённой настройке сводить к нулю или минимуму.

### Интерактивные элементы

- **Sticky nav с CTA** — доступна на любой глубине прокрутки
- **Floating CTA bar** — появляется снизу после скролла 30% (мобиль)
- **Tab switcher в demo-секции** — Финансы / Лайфстайл / Маркетинг
- **FAQ accordion** — по клику открывает ответ, остальные закрываются
- **Comparison table toggle** — "Показать полное сравнение ▾"
- **Smooth scroll** — все anchor-ссылки в nav, `scroll-behavior: smooth`

## 7. ТЕХНИЧЕСКИЕ ТРЕБОВАНИЯ

### Стек

**Вариант A (рекомендуется для старта):**
- HTML5 + CSS3 + Vanilla JS
- Никаких фреймворков — быстрее грузится, проще деплоить
- Статический хостинг: Netlify / Vercel / GitHub Pages

**Вариант B (если нужна масштабируемость):**
- Next.js (App Router) + Tailwind CSS
- Framer Motion для анимаций
- Vercel хостинг

### Производительность

- **Lighthouse Performance:** минимум 90 на мобильном
- **LCP (Largest Contentful Paint):** < 2.5s
- **CLS (Cumulative Layout Shift):** < 0.1
- **Изображения:** WebP формат, `loading="lazy"` для всего ниже fold
- **Шрифты:** `font-display: swap`, preload для hero шрифта
- **Анимации:** `will-change: transform` только на активно анимируемых элементах

### Адаптивность (Mobile-First)

```
Breakpoints:
375px  — base (iPhone SE, основной таргет)
428px  — large mobile
768px  — tablet
1024px — desktop small
1280px — desktop
1440px — desktop large (max content width)
```

**Приоритет верстки:** Сначала 375px, потом расширяем. Не наоборот.

**Mobile-специфичные требования:**
- Все tap targets минимум 44×44px
- Нет hover-only функциональности
- Bento grid → 1 колонка
- Phone mockup → уменьшается пропорционально
- Pricing → вертикальный стак, featured карточка первая

### Доступность

- Contrast ratio 4.5:1+ для всего body text (WCAG AA)
- Все изображения: `alt` атрибуты
- FAQ accordion: ARIA `aria-expanded`, `aria-controls`
- Keyboard navigation: focus visible на всех интерактивных элементах
- `prefers-reduced-motion`: отключать или минимизировать все анимации

### SEO

```html
<title>Copy-Bot — Пост для Telegram-канала за 3 минуты</title>
<meta name="description"
  content="AI-бот генерирует готовые посты для Telegram прямо в мессенджере.
  10+ форматов, расписание публикаций, оплата Stars. 3 поста бесплатно.">
<meta property="og:title" content="Copy-Bot — Посты для Telegram за 3 минуты">
<meta property="og:image" content="/og-image.png">  <!-- 1200×630px -->
```

### Метрики и аналитика

- Яндекс.Метрика (основная — русскоязычная аудитория) или Plausible
- Отслеживать события:
  - `cta_click` (hero / pricing / final) с параметром `location`
  - `pricing_tab_view` (какой тариф смотрели)
  - `faq_open` (какой вопрос)
  - `demo_tab_switch` (какую нишу выбрали)
  - `scroll_depth` (25%, 50%, 75%, 100%)

## 8. ЧЕГО ИЗБЕГАТЬ

### Контент и копирайтинг
- ❌ Слово "платформа" — слишком корпоративно для Telegram-бота
- ❌ "AI-powered content generation" — английский шаблонный маркетинг

- ❌ Generic testimonials без @handle и количества подписчиков
- ❌ "Быстро", "удобно", "мощно" без конкретных цифр
- ❌ Скрытые условия "звёздочкой" под pricing

### Дизайн
- ❌ Стоковые фото (ноутбук на столе, улыбающийся человек)
- ❌ Белый фон + синяя кнопка (SaaS-шаблон 2018)
- ❌ Particle.js / floating dot networks (устарело)
- ❌ Glassmorphism везде (только nav + 1 focal элемент)
- ❌ Агрессивный parallax (укачивает, убивает Lighthouse)
- ❌ Анимация каждого элемента страницы (хаотично)

- ❌ Поп-ап в первые 30 секунд (любой)
- ❌ Автозапуск видео/аудио
- ❌ Cookie banner на пол-экрана мобильного (перекрывает CTA)

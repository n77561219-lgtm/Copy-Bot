# Copy-Bot — коротко о продукте

**Copy-Bot** — Telegram-бот, который пишет посты для каналов и сообществ.

Автор называет тему — бот за 30 секунд возвращает готовый текст, отформатированный под Telegram, ВКонтакте или MAX. Один запрос — три варианта под три площадки.

---

## Что умеет

- Генерирует посты в 10+ форматах: экспертный, сторителлинг, список, провокация, кейс и другие
- Адаптирует формат под площадку: короткие абзацы для Telegram, хэштеги для ВКонтакте, лаконичный заголовок для MAX
- Учитывает тон канала — можно загрузить примеры своих постов
- Предлагает темы по нише, если идей нет
- Ставит посты в очередь с расписанием по дням и времени
- Работает полностью внутри Telegram, оплата — картой через ЮКасса

## Для кого

Авторы Telegram-каналов, администраторы сообществ ВКонтакте, SMM-менеджеры, которые ведут несколько аккаунтов. Те, кто знает о чём писать, но тратит слишком много времени на «как».
